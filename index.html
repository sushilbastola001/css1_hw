Margin with one value

Margin with one value is specified if you want same margin specification in all sides.

This is for padding with one value.

Margin with two value

Margin with two value is specified if you want different margin specification for Top & Bottom and Left & Right sides. First value for Top & Bottom and second value for Left & Right side.

This is for padding with two value.

Margin with three value

Margin with three value is specified if you want different margin specification for Top, Left&Right, and Bottom sides. First value is for Top, second value is for Left & Right and third value is for Bottom side.

This is for padding with three value.
